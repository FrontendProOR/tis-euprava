# tis-euprava
Fakultetski timski projekat za predmet Tehnologije i sistemi eUprave.

Zahtevi za izradu projekta su sledeći:
   - svaki član tima mora da ima poseban sistem
   - na nivou celog tima potrebno je da imate poseban sistem sa korisnicima i Single sign-on (SSO) metodu autentifikacije
   - koristite API za komunikaciju između servisa
   - svaki član tima treba da istraži i osmisli funkcionalnosti unutar svog servisa (minimalno 5 zahteva isključujući CRUD operacije)
   - neophodno je imati barem tri razmene podataka između servisa
